# Fluxus-Key-Bypasser

This Ruby script bypasses the Fluxus key system and retrieves the key in just 2 seconds.

## Prerequisites

Before running the script, make sure you have the following gems installed:

```bash
gem install nokogiri
gem install artii
ruby main.rb
```
